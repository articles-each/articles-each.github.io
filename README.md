articles.each {
============================

_articles.each {_ は __プログラミング関連の記事限定__ のレコメンデーションアプリです。あなたの好みに合わせて、毎日数本の記事を配信します。配信された記事を評価することで、より適切な記事が配信されるようになります。

プラットフォーム
----------------------------

- iOS (8.0 or later)
- Android (Comming soon)
